# Published Systematic Review and Meta-analysis (RevMan)

## Overview
This repository contains data and analysis files related to a **published systematic review and meta-analysis** conducted using Review Manager (RevMan 5).

## Publication
This work has been published in:
- **Journal:** Journal of Obstetrics and Gynecology of India
- **Year:** 2022
- **DOI:** https://doi.org/10.1007/s43032-022-00933-y

The publisher-formatted article is not hosted here due to copyright restrictions.

## Software Used
- Review Manager (RevMan) version 5
- Random-effects model used where applicable

## Repository Contents
- `revman/` – RevMan project file (.rm5)
- `citation/` – Publication citation details

## Reproducibility
All analyses can be reproduced by opening the `.rm5` file in Review Manager (RevMan).
